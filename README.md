# To-Do List Tracker

This is a simple web-based to-do list application built with React.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/todo-list-tracker.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

### Usage

To start the development server, run:

```sh
npm start
```

This will open the app in your browser at [http://localhost:3000](http://localhost:3000).
